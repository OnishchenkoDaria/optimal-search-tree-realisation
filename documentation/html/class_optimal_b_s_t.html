Detailed Description

Class representing an Optimal Binary Search Tree (BST).

The OptimalBST class includes methods for calculating and constructing an optimal BST using both sequential and parallel approaches. It uses a Singleton pattern to ensure a single instance of the class is created and provides thread-safe operations.
