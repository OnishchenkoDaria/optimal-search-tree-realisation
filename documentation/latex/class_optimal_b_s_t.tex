\doxysubsection{Detailed Description}
Class representing an Optimal Binary Search Tree (BST). 

The \doxylink{class_optimal_b_s_t}{Optimal\+BST} class includes methods for calculating and constructing an optimal BST using both sequential and parallel approaches. It uses a Singleton pattern to ensure a single instance of the class is created and provides thread-\/safe operations.
